# Should You Buy? — Singapore Property Wealth Projection Tool

A free, open-source wealth projection tool for Singaporeans comparing staying put, renting, or buying property. Built by Leonard, a regular Singaporean figuring out finances like the rest of us.

## What it does

Many people believe Singapore property is the best way to build wealth. Maybe, maybe not. This tool helps you project your wealth based on how you split it between equities, bonds, and specific Singapore properties, using historical averages and your actual numbers.

Compare staying with parents and investing, renting and investing the difference, buying a resale HDB, or buying a private condo. Each path is simulated year by year with salary growth, expense inflation, rent escalation, mortgage payments, CPF, property appreciation with lease depreciation, and a blended investment portfolio.

## Key features

Choose from popular ETFs (VWRA, IWDA, CSPX, ES3) with real 10-year returns. Separate equity and bond return inputs. Side-by-side comparison matrix. Year-by-year wealth chart showing all paths including unaffordable ones as dashed lines. Recommends the best affordable path. Quick-adjust on results page for instant recalculation. Share results (without personal data) or save a link with full inputs. Sensitivity analysis. Concentration vs diversification risk explanation. Buy-to-rent scenario support. CPF housing grants. TDSR/MSR checks. Liquid vs non-liquid wealth breakdown.

Runs entirely client-side. No data sent anywhere. Open source.

## Technical details

Single HTML file, no dependencies, no build step. Vanilla JS with custom canvas charting. MIT licensed.
